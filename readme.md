# readme

# Dependency

Install [Hugo](https://gohugo.io/)

# Usage

```bash
git clone https://github.com/Scratch3Lab/codelab-www
cd codelab-www
hugo server --theme=airspace-hugo --buildDrafts # open http://localhost:1313/
```
